### Features
 
#### Launch
- Runs as a single file
- Not required server
- Working in LAN/VPN
- Manual configuration of receiver's IP/port

#### Audio capturing
- System mic is used
- Audio format: PCM, stereo, 16-bit, fixed sample rate (44.1 kHz)

#### Audio transmission
- UDP
- 1024 byte payload
- Realtime transmission

#### Receiving and playing
- Realtime audio playing
